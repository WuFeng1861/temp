# 骰子赌场游戏

一个基于 Vue 3 + Spring Boot 3 开发的在线骰子赌场游戏。

## 功能特点

- 3D骰子动画效果
- 用户注册登录
- 实时余额显示
- 游戏历史记录
- JWT身份验证
- Swagger API文档

## 技术栈

### 前端
- Vue 3
- Pinia
- Three.js
- TailwindCSS
- Element Plus
- Axios

### 后端
- Spring Boot 3
- Spring Security
- Spring Data JPA
- H2 Database
- JWT
- Swagger/OpenAPI
